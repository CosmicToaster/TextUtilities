# TextUtilities

A sporty, minimal Markdown ↔ Rich Text translator. Paste markdown and copy formatted rich text to your clipboard — or go the other way around.

## Features

- **Markdown → Rich Text**: Type or paste markdown, copy as formatted rich text for emails, docs, etc.
- **Rich Text → Markdown**: Paste rich text, get clean markdown output
- **Clipboard-first**: One-click copy with modern Clipboard API + fallback support
- **Responsive**: Works on desktop and mobile

## Deploy

Import this repository on [Vercel](https://vercel.com) — zero configuration needed. The `vercel.json` is already set up.

## Local Development

```bash
npx serve public
```

Then open [http://localhost:3000](http://localhost:3000).
